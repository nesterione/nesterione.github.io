Courcera.org замечательный ресурс для людей которые хотят научиться чему-то новому. В последнее время я прохошу большое кличество курсов. Многие только для ознакомления, но вот машинным обучением занимаюсь более серьезно.

Для тех кому тоже интересна эта тема, советую пройти курсы из специализации "Наука о данных". Там 9 промежуточных курсов, каждый из которых длится 4 недели, программировать придется на языке R (там есть соответствующий курс)

Я завершил очередной курс из серии, могу отметить качество организации курса. Информация дается в доступнй форме, есть субтитры на многих языках, достаточно практики.
