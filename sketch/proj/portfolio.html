Верстка страниц:

Описание:

Примером адаптивной верстики может служить данный сайт. Что такое адаптивная верстка? Адаптивный сайт хорошо отображается на устройствах с разничными экранами. Например телефоны, планшеты.

Можете уменьшить окно браузера, и Вы увидите, что отображение данных на странице меняется в зависимости от ширины.

Использованные технологии:

  • чистый HTML5/CSS3 (без фреймворков)
  • чистый JavaScript (без jQuery)

Описание:

Данный проект служит примером классической верстки страницы по макету (картинке).

Как правило Вы даете свое изображение, мы делаем вам HTML страницу, которая выглядит в браузере также как на изображении. Созданный макет Вы с легкостью можете наполнить своей, актуальной информацие и получить сайт в соответствии с Вашими желаниями.

Использованные технологии:

  • чистый HTML5/CSS3 (без фреймворков)

Описание:

Большинство страниц предназначены для получения от пользователя каких-либо данных, для этого на Вашем сайте нужна форм для ввода данных.

Некоторые формы имеют сложную структуру, в данном случае представлена статическая форма, но при необходимости может быть добавлена валидация на стороне клиента. В таком случае если пользователь введен некорректные значения (например e-mail), то он получит оперативно (без лишних обращений к серверу) соответствующее уведомление.

Вы получаете благодантых клиентов, которые не устанут пользоваться Вашим сайтом.
