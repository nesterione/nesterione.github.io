# nesterione.com

Personal site for Ihar Nestsiarenia. Bilingual (RU primary, EN secondary), built with [Astro](https://astro.build), deployed to GitHub Pages from `master` via the workflow in `.github/workflows/deploy.yml`.

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # ./dist
```

Notes live in `src/content/posts/{ru,en}/<slug>.md`. See `CLAUDE.md` for the full editing playbook.
